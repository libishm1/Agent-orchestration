## Rules

- Raw is immutable. Do not edit existing files.
- Add new files only. Use the metadata header below.
- Do not delete files. If a file is wrong, leave it and add a corrective note in the relevant wiki page.

## File header

## How to ingest

After adding files here, run `/ingest_raw local_ai_codex_claude` (Claude Code) or paste the Codex equivalent prompt with the topic name. The wiki-curator will summarize into the matching `wiki/local_ai_codex_claude/` pages.
