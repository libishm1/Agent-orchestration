# raw/ur10e_ros2_grasshopper/

UR10e + RG6 + ROS2 + Grasshopper integration. Pose conversion notes, ur_rtde session logs, ROS2 driver setup logs, URScript samples, OnRobot URCap manual extracts, gripper firmware notes, calibration measurements.

## Rules

- Raw is immutable. Do not edit existing files.
- Add new files only. Use the metadata header below.
- Do not delete files. If a file is wrong, leave it and add a corrective note in the relevant wiki page.

## File header

Every new raw file in this folder uses this header:

```markdown
# Title

Date:
Source:
Topic: ur10e_ros2_grasshopper
Status: raw
Do not edit: true
```

## How to ingest

After adding files here, run `/ingest_raw ur10e_ros2_grasshopper` (Claude Code) or paste the Codex equivalent prompt with the topic name. The wiki-curator will summarize into the matching `wiki/ur10e_ros2_grasshopper/` pages.
