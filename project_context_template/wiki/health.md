## How to read this file

Every issue is tagged with priority:

- **P0** — must fix. Contradiction in confirmed facts, broken link in index, missing safety-critical info.
- **P1** — should fix. Orphan page, stale API claim, undefined glossary term.
- **P2** — nice to fix. Stylistic inconsistency, minor structural cleanup.

Run `/health_check_wiki` to refresh. The health-checker subagent updates this file.
